# Results

This folder contains the numerical outputs, acceptance checks, restart files, tables, and figures generated by the five notebooks.

## Folder structure

- `packing/` – initial particle-packing outputs
- `compression/` – vertical compression results
- `shear/` – direct-shear results
- `pore_pressure/` – pore-pressure sensitivity results
- `fem_dem/` – FEM-to-DEM comparison and stress mapping
- `restart/` – saved particle configurations used between notebooks
